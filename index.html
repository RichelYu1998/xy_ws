功能特性

商品爬虫

自动登录szwego.com，支持Cookie保存和加载，智能滚动加载所有商品信息

货号对比

支持交互式和简化版两种模式，自动检测重复序列号，详细对比结果展示

Excel支持

直接读取Excel文件，智能工作表识别，精确列读取，保留前导0

并发处理

多线程并发处理提高采集效率，自动提取商品名称、价格、货号、备注、员工信息

跨平台支持

完美支持Windows、Mac、Linux系统，统一的配置和运行方式

反爬虫规避

内置反爬虫检测和规避机制，智能请求频率控制，降低被封禁风险

使用方法

环境要求： Python 3.8+, pip

1. 安装依赖

# 创建虚拟环境
python -m venv .venv

# 激活虚拟环境
# Windows:
.venv\Scripts\activate
# Linux/Mac:
source .venv/bin/activate

# 安装依赖
pip install playwright openpyxl
playwright install chromium

2. 配置文件

编辑 config/config.json 文件：

{
  "login": {
    "username": "your_username",
    "password": "your_password",
    "login_type": "phone"
  },
  "target_url": "https://www.szwego.com/your_shop_url",
  "output_file": "file/output.json",
  "cookie_file": "config/cookies.json",
  "excel_file": "path/to/your_file.xlsx"
}

3. 运行程序

根据您的操作系统选择对应的命令：

macOS / Linux: python3 main.py
Windows: python main.py

4. 货号对比

程序支持两种模式进行货号对比：

  • 交互模式 - 手动输入货号进行对比
  • 文件模式 - 从JSON或Excel文件读取货号进行对比
  • 差异日志 - 自动保存对比差异到按日期命名的日志文件
